# Marketing_Data_Sales

1. Project Title / Headline
Global Sales & Marketing Performance Analytics Dashboard
Subtitle: An end-to-end data pipeline and interactive visualization for multi-regional retail performance.

2. Short Description / Purpose
This project transforms raw transactional data into actionable business intelligence. By integrating sales metrics across five global regions, the dashboard identifies high-value customer segments, monitors product category performance, and analyzes discount impacts on net profitability. It serves as a centralized tool for stakeholders to track real-time KPIs and optimize regional marketing strategies.

3. Tech Stack
Data Processing: Python (Pandas, NumPy) for data cleaning and handling null values.

Database Management: SQL for querying and joining multi-table relational data.

Visualization: Power BI (DAX, Power Query) for interactive reporting.

Data Modeling: Star Schema architecture to optimize performance for large datasets.

4. Data Source
Dataset: A comprehensive sales and marketing dataset containing over 25,000 rows.

Attributes: Includes Order ID, Order Date, Customer Demographics, Geographical Data (Country, State, Region), Product Categories (Appliances, Electronics, Furniture, etc.), and Financial Metrics (Unit Price, Discount %, Net Sales).

5. Features / Highlights for GitHub
Complex Data Modeling: Managed relationships between multiple sales tables to ensure accurate data granularity across 12 months.

Advanced DAX Calculations: Implemented custom measures for Net Sales, Discount Impact, and Year-over-Year (YoY) growth.

Interactive Slicers: Enabled dynamic filtering by Region, Category, and Time Period for deep-dive analysis.

Profitability Analysis: Visualized the correlation between discount percentages and net sales to identify the "sweet spot" for promotional offers.

Geographical Mapping: Integrated map visuals to highlight top-performing states and countries at a glance.
